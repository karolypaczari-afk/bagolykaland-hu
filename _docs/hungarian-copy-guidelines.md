# Hungarian Copy & Tone Guidelines — BagolykaLand

A BagolykaLand szövegeinek mindig **közvetlen, empatikus, természetes magyarul** kell szólniuk — soha „AI-ízű", nyersfordításos, vagy hivataloskodóan szervetlen. Ez a dokumentum a 2026-os iskola-előkészítő landoló-redesign közben kicsiszolt elveket rögzíti. **Bármely új landoló / blog / email / form-copy írása előtt először ezt nézd át.**

Kapcsolódó dokumentum: a `hungarian-localization` skill (`~/.claude/skills/hungarian-localization/SKILL.md`) az általános magyar helyesírást fedi le; ez az itteni doksi a **BagolykaLand-specifikus stílus** réteget adja hozzá.

---

## 1. Alaphangulat (tone)

| Igen | Nem |
|---|---|
| **Tegeződés** ("Te / gyermeked / felhívunk") | Magázódás landing oldalon |
| Empatikus, „melletted állunk" hang | Lekezelő szakzsargon |
| Konkrét, gyakorlati ("3×30 perc, 8 héten át, 8–10 fő") | Absztrakt marketing-frázis |
| Direct CTA: konkrét akció | „Kattints ide!" típusú generic CTA |
| Esetenként rövid, megálló mondat ritmusa | Loncsos, hosszú összetett mondatok |
| Egy adott bekezdésen belül **konzisztens perspektíva** | „Mi" és „én" egy mondaton belül |

A célközönség: **3–10 éves gyerekek szülei**. A copy úgy szóljon, mint egy bizalmas óvodapedagógus / tapasztalt szakember, aki mellettünk áll és segít. Ne tudományoskodjon, de ne is fukarkodjon a szakszavakkal (a szülő tisztel ha látja a hozzáértést).

---

## 2. Kerülendő anglicizmusok (a leggyakoribb csapdák)

| Angol-tükör | Helyes magyar | Miért |
|---|---|---|
| specifikus | konkrét, célzott | A „specifikus" csak orvosi/szakmai precíz kontextusban OK; landingen idegen |
| kalkulációt csinál | sokat töprengsz, sokat számolsz, mérlegelsz | A „csinál + főnév" angol szerkezet |
| nővértestvér program | társprogram, kapcsolódó program | „Sister program" tükörfordítás |
| neked illeszkedik | neked alkalmas, neked passzol, amikor időd engedi | „Fits you" tükörfordítás |
| proaktív vagy | előrelátó vagy, időben lépsz | Az angol jövevény tegezős kontextusban beszédbe nem illik |
| pólpozícióból indul | felkészülten indul, jó kezdéssel | „Pole position" autósport-anglicizmus |
| opció | választás, lehetőség | „Option" tükörfordítás |
| szignifikáns | jelentős, érzékelhető | Csak statisztikai kontextusban OK |
| szupportál | támogat | Nyilvánvaló |
| eszkalálódik | súlyosbodik, fokozódik | Stilisztika |
| Felkészült pedagógiai munka | Alapos szakmai munka, gondos pedagógiai munka | „Felkészült munka" — felkészült pedagógus van, munka nem lehet „felkészült" |
| holisztikus megközelítés | átfogó megközelítés, komplex szemlélet | „Holistic" jövevény |
| optimalizálás-érzelemből | óvatlanul, kapkodva | Túl elvont |

**Magyar nyelvi felismerési teszt**: olvasd fel hangosan. Ha a kifejezés úgy hangzik, mintha kétnyelvű cégfolyosón hallanád, valószínűleg anglicizmus.

## 3. Aspektus, módkövetés, idő

### Feltételes mód: jövő ≠ múlt
- ❌ „Szeretnéd, ha **mert volna** jelentkezni" → feltételes MÚLT → „már lekésted" érzés
- ✅ „Szeretnéd, ha **merne** jelentkezni" → feltételes JELEN/JÖVŐ → még van rá lehetőség

A gyermekekről szóló JÖVŐBELI fejlődési kép soha ne legyen feltételes múltban.

### „Hogy" beszúrása
- ❌ „Félsz, az első hibánál kiszáll"
- ✅ „Félsz, **hogy** az első hibánál már baj lesz"

A magyar tegezős copy-ban a „hogy" gyakran kötelező az alárendelő mellékmondatok elején. Nyersfordításnál tipikusan elmarad.

### Igeidők keverése egy mondaton belül
- Tartsd jelen időben, vagy ha múltbeli eseményről beszélsz, konzisztensen múltban.

## 4. Természetes szófordulatok a magyar fejlesztőpedagógus szóhasználatban

(2026-os Tavily-kutatás alapján, az aranyoskozszolg.hu, athidalo.com, fokit.hu, fejlesztoiskola.hu, grofroberta.hu, holiplace.hu mintákból.)

| Természetes magyar | Mikor használd |
|---|---|
| „magabiztos iskolakezdés" | hero, CTA, závró |
| „zökkenőmentes / sikeres iskolakezdés" | sub-headline, value prop |
| „kudarctűrő képesség", „kudarctűrés" | szakmai bemutatás |
| „részképesség-fejlesztés" | szakszó (egybeírandó) |
| „egyénre szabott", „differenciált fejlesztés" | módszertan |
| „játékos, mozgásos, tevékenykedtető gyakorlatok" | módszertan-leírás |
| „figyelem, memória, finommotorika, szem-kéz koordináció" | készség-fejlesztés |
| „iskolaérettség" | egybeírandó, szakszó |
| „fokozatos átmenet az óvodai működésből az iskolai feladathelyzetekbe" | átmenet-magyarázat |
| „bemeneti / kimeneti mérés" | program-strukturálás |
| „írásos szülői elemzés" | dokumentáció említése |
| „kiscsoport (max 8–10 fő)" | csoportlétszám |
| „felkészített gyermek" | proof / track record |
| „idegen helyzetben elnémul" | szorongó gyermek leírása |
| „nem tud sokáig megülni egy helyben" | hiperaktív / mozgásigény |
| „nem ismeri fel a számokat 10-ig" | konkrét hiány |
| „nem fogja biztosan a ceruzát" | finommotorika-hiány |

### Kerülendő szakzsargon (túl kemény szülő-fülnek)
- ❌ „kognitív funkciók" → ✅ „gondolkodás, figyelem"
- ❌ „motoros koordináció" → ✅ „kéz-szem összerendezés", „mozgás összerendezése"
- ❌ „grafomotoros készségek" → ✅ „ceruzahasználat, kézírás-előkészítés"
- ❌ „nyelvi-kommunikatív kompetencia" → ✅ „beszéd, kommunikáció"

A szakszó akkor használandó, ha pontos jelentést hordoz (pl. „szerialitás"), de mellette **mindig** egy mindennapi magyarázat ("sorba rendezés, sorrendiség").

## 5. „Mi" vs „Én" perspektíva — egy oldalon belül konzisztens

A BagolykaLand többszereplős hang: brand-kollektíva (BagolykaLand csapata) ÉS Szabó-Varga Alexandra (alapító) együtt. Ezért:

| Kontextus | Hang | Példa |
|---|---|---|
| Általános brand-üzenet | **T/1 „mi"** | „Felhívunk 48 órán belül", „segítünk eldönteni" |
| FAQ válaszok | **T/1 „mi"** | „Maximum 8–10 fős kiscsoportban dolgozunk" |
| Footer / reassure-strip | **T/1 „mi"** | „Visszafizetési garancia — díja visszajár" |
| Alexandra **közvetlen** szólal meg (videó eyebrow, founder quote, „Hívj fel — segítek…") | **E/1 „én"** | „Elmondom, hogyan készítjük fel", „Segítek személyesen eldönteni" |

**Ne keverd egy mondaton belül.** Vagy „Hívj fel — segítek személyesen" vagy „Hívj fel — segítünk személyesen". Az első csak akkor jó, ha Alexandra-context van (telefonon hozzá hív, ő veszi fel). A második semleges brand-hang.

## 6. CTA copy — gombok, microcopy

| Régi (kerülendő) | Új (preferált) |
|---|---|
| Foglalom a helyem | **Lefoglalom a helyem** |
| Inkább telefonon kezdesz? | **Egyeztessünk telefonon?** |
| Klikkelj ide / Kattints! | Konkrét akció: „Megnézem", „Olvasom", „Visszahívást kérek" |
| Küldés... (form pending) | „Küldés…" (3 pont „…" karakter, nem 3 ASCII pont) |
| Submit / Send | „Bebiztosítom a helyem", „Visszahívást kérek", „Elküldöm" |

### Form labels
- „Szülő neve" — nem „Név"
- „E-mail cím" — kötőjellel, nem „Email cím"
- „Telefonszám" — egybe
- „Gyermek életkora (évben)" — placeholder

### Phone reassurance pattern (a BagolykaLand sztenderdje)
```
📞 Egyeztessünk telefonon?
Hívd Alexandrát közvetlenül: +36 30 364 9396
Hétfő–péntek 9:00–18:00 · Ha nem érünk rá azonnal, hagyj üzenetet — visszahívunk.
```

## 7. Hormozi-jellegű marketing-mechanika MAGYARRA átültetve

A direct response marketing eszközöket (scarcity, proof, value stack, guarantee) **lefokozott, magyarosan visszafogott** változatban használjuk. Soha nem amerikai stílusú „HURRY UP! ONLY 3 LEFT!" hang.

| Eszköz | Magyaros megfogalmazás |
|---|---|
| Scarcity | „Csak 8–10 férőhely turnusonként" / „Korlátozott férőhely — a jelentkezési sorrend dönt" |
| Proof / Stat | „1000+ felkészített gyermek · 15+ év · saját módszertan · max 8–10 fős csoport" |
| Stakes | „Mi történik, ha nem készíted fel időben?" + 4 konkrét következmény (pénzben mérve is) |
| Risk reversal | „Bemeneti mérés után döntesz — az első alkalom díja visszajár, ha úgy érzed, nem ez kell" |
| Value stack | „Ha ezeket egyenként rendelnéd meg, ennyibe kerülnének külön-külön. Csoportos programként:..." |
| Authority | Founder long-form blockquote, fact-lista („15+ év, 1000+ gyermek, saját strukturált módszertan") |

**A magyar szülő érzékeny a túltolt, amerikás CTA-szövegre.** A copy-nak mindig kell egy „érzelmi szoba" a marketing-mechanika mellett — egy empatikus mondat, egy konkrét sztori, egy „nem vagy egyedül" gesztus.

## 8. Pénzformátum, dátum (CLAUDE.md emlékeztető)

- Ezres elválasztó: **pont** → `10.000 Ft`, `~252.000 Ft`, `28.000 Ft / hó`
- Tizedesjel: **vessző** → `3,5 év`
- Dátum: `2026. szeptember 1.` (hónap kis kezdőbetű, év után pont, nap után pont)
- Időpont: `9:00–18:00`
- Idézőjel: `„…"` (magyar tipográfiai, `&bdquo;` + `&rdquo;` entitások); YAML-front matter-ben kerülni VAGY single-quote bezáró stringben

---

## 9. Em-dash konvenció (brand-specifikus)

A `hungarian-localization` skill ajánlja az en-dash `–`-et közbevetéshez. A BagolykaLand **brand-konzisztencia okán** **em-dash** `—`-et használ (`&mdash;` entity), szóközökkel:

- ✅ „Magabiztosabb iskolakezdés &mdash; kudarcélmények nélkül"
- ❌ „Magabiztosabb iskolakezdés – kudarcélmények nélkül" (en-dash)

Ez tudatos brand-stílus, ne változtass rajta.

## 10. Mintapéldák — a redesign tanulságai

### Persona kártya rosszul → jól
- ❌ „Kudarc-érzékeny a gyermeked — félsz, az első hibánál kiszáll"
- ✅ „A gyermeked érzékeny a kudarcra — félsz, hogy az első hibánál már baj lesz"
  - *Természetesebb szórend, „hogy"-kötőszó kötelező, „baj lesz" konkrétabb mint „kiszáll"*

### Szorongó gyermek persona
- ❌ „szeretnéd, ha mert volna jelentkezni" (feltételes múlt)
- ✅ „szeretnéd, ha bátrabban szólalna meg" (feltételes jelen + konkrét cselekvés)

### Gyenge területek
- ❌ „Konkrét gyengébb területeket akarsz fejleszteni" (rossz szórend, defektorientált)
- ✅ „Egy konkrét részképességet szeretnél megerősíteni" (pozitív megfogalmazás)

### Sulirajt cross-promo
- ❌ „A nővértestvér programunk: akkor csinálod, amikor neked illeszkedik"
- ✅ „A társprogramunk: akkor foglalkozol vele, amikor neked alkalmas"

### Telefonos reassurance
- ❌ „Ha most nincs ideje, üzenetet hagyhatsz — visszahívjuk." (kétértelmű alany)
- ✅ „Ha nem érünk rá azonnal, hagyj üzenetet — visszahívunk." (egyértelmű, természetes)

### Záró CTA
- ❌ „nem kell bonyolult kalkulációkat csinálnod"
- ✅ „nem kell sokat töprengened rajta"

## 11. Mielőtt push-olnál — copy-checklist

1. [ ] Olvasd fel **hangosan** a szöveget magadnak. Ha sehol nem akad meg a nyelved, valószínűleg magyaros.
2. [ ] Keress anglicizmust (a #2-es táblázat alapján): „specifikus", „kalkuláció", „opció", „proaktív" stb.
3. [ ] Konzisztens-e a perspektíva (mi vs én) egy bekezdésen belül?
4. [ ] Helyes-e a pénz- és dátumformátum?
5. [ ] Em-dash szóközökkel, nem en-dash?
6. [ ] „BagolykaLand" minden előfordulása helyesen?
7. [ ] A CTA-k konkrét akciót jelölnek (nem „klikk ide")?
8. [ ] Az aspektus-mód-idő stimmel (nincs feltételes múlt jövőbeli kontextusban)?
9. [ ] Az adott szülő-archetípusra (persona) szóló mondat valódi szülő-szóhasználat-e? (Ha mintaként olvashatnád egy szülői csoport Facebook-posztjában, jó. Ha PR-szövegnek hangzik, rossz.)
10. [ ] Az érzelmi terhelés és a marketing-mechanika kiegyensúlyozott? (Nincs túltolva, sem érzéketlen.)

## Frissítés-protokoll

Ha új landoló-iteráció során további szófordulatokat csiszolunk ki:
1. Add hozzá a #10-es „Mintapéldák" szekcióhoz a `❌ → ✅` formátumban.
2. Ha új anglicizmus-kerülendőt találunk, a #2-es táblázatba.
3. Commit-üzenetben legyen `docs(copy-guidelines): ...` prefix.

Ez a dokumentum élő gyűjtemény — minden konverziójavító copy-iteráció után tükröznie kell a tanultakat.
